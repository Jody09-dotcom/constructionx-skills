# UK Site Diary Guide

Standard fields expected on a UK construction site diary. Informed by CITB site manager training and common main-contractor practice.

## Purpose

A site diary is the contemporaneous daily record of what happened on site. It is used for progress tracking, claim evidence (delays, variations, loss and expense), subcontractor coordination, and audit. Kept properly, it is the single most valuable document in a dispute.

## Standard Field Set

| Field | Notes |
|-------|-------|
| Date | One entry per working day. Back-fill is allowed but must be marked. |
| Site name and reference | Must match the project agreement. |
| Site manager name | The person accountable for the entry. |
| Weather, am and pm | Conditions, temperature, wind. Relevant for delay claims. |
| Labour breakdown | By trade, direct vs subcontractor, totals. Basis for productivity analysis. |
| Plant and equipment | Owner (hire vs direct), in/out times. Basis for plant hire reconciliation. |
| Deliveries | Time, supplier, material, quantity. Cross-reference with delivery tickets. |
| Works completed | By trade and location. Short, specific. "Blockwork, 1st lift, plot 4 west elevation" beats "block work". |
| Delays | Reason, hours lost, follow-up action. The entry that matters most for any Extension of Time claim. |
| Incidents | Time, description, persons, action. Flag any RIDDOR reportable. |
| Near-misses | Treat the same as incidents. Cultural indicator as well as record. |
| Visitors | Name, company, time in/out, inducted y/n. |
| Photos | Count, subjects, consent status. |
| Free-text notes | Observations, items for tomorrow, instructions received. |
| Sign-off | Site manager signature and date. |

## UK GDPR, Photos of Workers

Workers retain data-protection rights on site. Photographing workers in a way that identifies them requires a lawful basis. In practice, this is consent (site induction consent form) or legitimate interest (documented). The diary should record whether photos taken that day include identifiable workers, and whether consent was obtained.

Group shots from a distance where individuals are not identifiable are lower risk. Close-up shots, especially of faces, are higher risk.

## RIDDOR, Separate Reporting

The diary is not a substitute for RIDDOR reporting. Reportable incidents (specified injuries, over-7-day incapacitation, dangerous occurrences, occupational diseases) must be reported to HSE at www.hse.gov.uk/riddor within the required timeframe. The diary entry notes that a RIDDOR report was made and references the RIDDOR reference number.

## Good Practice

- Write the diary on the day. End of shift is the golden window.
- Keep entries factual. Save opinion for a separate internal note.
- If you did not see it yourself, record who did.
- Numbers and times beat adjectives. "3 bricklayers" beats "a few bricklayers".
- Cross-reference: a delay entry should line up with a works-completed entry and a plant-idle entry.
- Store the diary in a location backed up daily. A diary only a site manager can access is half a diary.
